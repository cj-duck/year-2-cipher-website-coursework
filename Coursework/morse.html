Morse Code

Despite it's name, Morse Code is technically a cipher that turns a message into a sequences of "dots" and "dashes". It was developed in 1836 for use with electrical telegraph systems. Read more here.

A telegraph key like this one would be used to send morse code messages and were produced in huge quantities in WWII. The signal is "on" when the knob is pressed, and "off" when it is released. Length and timing of the dots and dashes are entirely controlled by the telegraphist.
